# 🚀 Employee Management System (EMS)

A modern **Employee Management System (EMS)** built using **Python, Flask, SQLite, Bootstrap 5, HTML, CSS, and JavaScript**. This project helps organizations manage employee records, attendance, payroll, reports, and administrative settings through a responsive web interface.

---

## 📌 Features

### 🔐 Authentication
- Admin Login
- Secure Logout
- Password Update

### 👨‍💼 Employee Management
- Add Employee
- Edit Employee
- Delete Employee
- Search Employee
- Sort & Pagination
- Employee Directory
- Avatar Support

### 📅 Attendance Management
- Mark Attendance
- Present / Absent / Late Status
- Attendance Records
- Delete Attendance
- Attendance Statistics

### 💰 Payroll Management
- Add Payroll
- Bonus & Deduction
- Automatic Net Salary Calculation
- Payroll Records
- Delete Payroll
- Payroll Summary

### 📊 Reports & Analytics
- Employee Statistics
- Department Summary
- Salary Summary
- Payroll Analytics
- Attendance Analytics
- Interactive Charts (Chart.js)

### ⚙️ Settings
- Admin Profile
- Change Password
- Company Information

### 🎨 UI Features
- Responsive Design
- Bootstrap 5
- SweetAlert2 Notifications
- DataTables Integration
- Chart.js Graphs
- Professional Dashboard
- Modern Sidebar Navigation

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/himanshu545/Employee-Management-System-Web.git
```

### Navigate to Project

```bash
cd Employee-Management-System-Web
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
